# Simplest-cpp-plot
Simplest method to build line plot (or other graphics, by modifiing basic functions) in console.
```c++
	plot(12, 20, x, y1, "pink");

	plot(12, 20, x, y2, "lime");

	plot(12, 20, x, y3, "violet");

	plot(12, 20, x, y4, "orange");
```
